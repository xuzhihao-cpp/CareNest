## 维护约束

- 家属端和护理端后续新增一级功能时，应继续使用现有顶部身份栏和固定底部导航。
- 移动端文字按钮建议保持至少 40px 点击高度，主要操作保持至少 44px。
- 新页面不得产生文档级横向滚动；表单和网格必须使用 `min-width: 0` 与响应式列。
- OpenAPI 契约变化后必须执行 `pnpm contract:generate`，并以 `pnpm contract:check` 验证生成物。
- Docker 网关集成脚本的默认地址已包含 `/api/v1`，不要把基址覆盖为站点根地址。
